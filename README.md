# CodeLab Exercises

Репозиторий с задачами по C# для платформы CodeLab. Каждое задание привязано к конкретной дате и хранится как отдельная единица в файловой системе.

## Что внутри

- `exercises/` — папка с упражнениями. Каждое упражнение хранится в своей директории (например, hello-world, find-max и др.; новые задания добавляются по мере необходимости)
- `exercises.json` — индекс всех упражнений, генерируется из файлов `meta.json`
- `scripts/` — вспомогательные PowerShell-скрипты для добавления и обновления упражнений

## Структура одного упражнения

Каждое упражнение содержит:

- `meta.json` — метаданные:
  - `slug` — уникальный идентификатор папки
  - `title` — название упражнения
  - `assignedDate` — дата назначения
- `instructions.md` — текст задания и условия
- `<PascalCase>.cs` — шаблон класса для решения
- `<PascalCase>Tests.cs` — тесты для проверки решения

## Как добавить новое упражнение

В папке `scripts/` есть скрипт `add-exercise.ps1`, который создаёт новую директорию с метаданными и шаблонами.

Пример команды PowerShell из корня репозитория:

```powershell
.\scripts\add-exercise.ps1 -Slug "new-exercise" -Title "New Exercise" -AssignedDate "2026-04-08"
```

После создания нового упражнения `exercises.json` обновится автоматически.

## Обновление индекса

Если вы вручную добавляете или изменяете папки упражнений, выполните:

```powershell
.\scripts\rebuild-config.ps1
```
