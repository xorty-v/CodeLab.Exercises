# Hello World

Добро пожаловать в первое задание проекта CodeLab! Это небольшое введение в мир C# и то, как здесь работают методы.

C# — это современный язык программирования от Microsoft. Он широко используется для создания приложений, сервисов и игр. В C# есть понятная структура, и одна из самых ранних задач — вывести на экран простое приветствие.

В программировании есть традиция: первая программа называется «Hello World». Это простой способ проверить, что среда работает и вы понимаете, как вывести текст.

В этом упражнении у вас есть статический класс `HelloWorld` и статический метод `SayHello`.

# Задание

Реализуйте метод `SayHello` в файле `HelloWorld.cs`, чтобы он возвращал строку:

```text
Hello CodeLab!
```

Тесты проверяют, что метод `HelloWorld.SayHello()` возвращает именно эту строку.
