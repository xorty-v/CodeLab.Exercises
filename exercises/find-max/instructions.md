# Find Max

Дан массив целых чисел `numbers`. Реализуй метод `GetMax`, который возвращает максимальный элемент массива.

Массив гарантированно не пустой, поэтому дополнительная проверка на пустоту не требуется.

## Примеры

```text
Input: numbers = [1, 3, 2]
Output: 3
```

```text
Input: numbers = [-5, -1, -7]
Output: -1
```
